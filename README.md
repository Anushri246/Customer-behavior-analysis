   # Customer Shopping Behavior Analysis


## Workflow

### 1. Data Preparation
* **Data Loading & Exploration:** Imported the dataset using Python and Pandas.
* **Data Cleaning:** Imputed missing review ratings using median and removed redundant information like promo code column

### 2. Database Analysis
* **SQL Integration:** Loaded the cleaned data into **PostgreSQL** for advanced analysis.
* **Key Finding:** Male customers generated **$157,890** compared to female customers' **$75,191**.

### 3. Visualization
* **Power BI Dashboard:** Created an **interactive interface** with dynamic filters.
* **Metrics:** Analyzed **3,900** transactions to identify revenue trends.
